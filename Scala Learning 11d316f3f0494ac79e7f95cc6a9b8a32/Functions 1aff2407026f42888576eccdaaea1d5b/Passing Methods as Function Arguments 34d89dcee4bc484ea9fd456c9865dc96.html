Passing Methods as Function Arguments

Passing Methods as Function Arguments in Scala

In Scala, you can pass methods as arguments to higher-order functions, treating methods as first-class citizens. This enables more flexible and reusable code structures. Here are some examples illustrating this concept:

Example 1: Applying a Mathematical Operation

def applyOperation(a: Int, b: Int, operation: (Int, Int) => Int): Int = {
  operation(a, b)
}

def add(x: Int, y: Int): Int = x + y
def subtract(x: Int, y: Int): Int = x - y

println(applyOperation(10, 5, add))      // Output: 15
println(applyOperation(10, 5, subtract)) // Output: 5

Example 2: Using Predefined Functions

def processList(numbers: List[Int], func: Int => Int): List[Int] = {
  numbers.map(func)
}

println(processList(List(1, 2, 3), Math.sqrt)) // Output: List(1.0, 1.4142135623730951, 1.7320508075688772)

Example 3: Filtering with a Custom Predicate

def filterStrings(strings: List[String], predicate: String => Boolean): List[String] = {
  strings.filter(predicate)
}

def isLong(s: String): Boolean = s.length > 5

val result = filterStrings(List("apple", "banana", "cherry"), isLong)
println(result) // Output: List(banana)

Example 4: Combining Functions

def combineFunctions(f: Int => Int, g: Int => Int): Int => Int = {
  x => f(g(x))
}

def double(x: Int): Int = x * 2
def increment(x: Int): Int = x + 1

val combinedFunction = combineFunctions(double _, increment _)
println(combinedFunction(5)) // Output: 12 (because (5 + 1) * 2 = 12)

Example 5: Using Anonymous Functions

def greetPerson(greeting: String => String, name: String): String = {
  greeting(name)
}

val formalGreeting = (name: String) => s"Hello, Mr. $name!"
println(greetPerson(formalGreeting, "Smith")) // Output: Hello, Mr. Smith!

Passing Methods as Function Arguments

This feature is often used to pass methods as arguments to higher-order functions, which expect function values:


def applyFunction(f: Int => Int, x: Int): Int = f(x)

def square(n: Int): Int = n * n

println(applyFunction(square _, 5)) // Output: 25

Conclusion

Passing methods as function arguments enables a higher level of abstraction and reusability in Scala code. It allows you to write more modular, flexible, and concise code by leveraging higher-order functions and treating methods as first-class citizens.
