Functions

Functions in Scala (Version 2.12.x)

Functions are fundamental building blocks in Scala. They allow you to encapsulate a piece of code, make it reusable, and abstract over behavior. Below, we will explore different aspects of functions in Scala, specifically targeting version 2.12.x.

1. Defining Functions

A function is defined using the def keyword, followed by the function name, parameters, return type, and body.

Example: Basic Function

def greet(name: String): String = {
  "Hello, " + name + "!"
}
println(greet("Alice")) // Output: Hello, Alice!

2. Parameterless Functions

Functions without parameters can be defined without parentheses.

Example

def currentTime: Long = System.currentTimeMillis()
println(currentTime) // Prints the current time in milliseconds

3. Default and Named Parameters

Functions can have default values for parameters, and you can use named arguments when calling a function.

Example

def printInfo(name: String, age: Int = 30): Unit = {
  println(s"Name: $name, Age: $age")
}
printInfo("John")            // Uses default age
printInfo(name = "Alice", age = 25) // Named parameters

4. Variable Arguments (Varargs)

A function can accept a variable number of arguments of the same type.

Example

def sum(numbers: Int*): Int = numbers.sum
println(sum(1, 2, 3, 4)) // Output: 10

5. Higher-Order Functions

Functions that take other functions as parameters or return functions are called higher-order functions.

Example

def applyTwice(f: Int => Int, x: Int): Int = f(f(x))
println(applyTwice(_ + 1, 5)) // Output: 7

6. Anonymous Functions (Lambdas)

You can define functions without a name, known as anonymous functions or lambdas.

Example

val add = (x: Int, y: Int) => x + y
println(add(3, 4)) // Output: 7

7. Partially Applied Functions

You can fix a certain number of arguments for a given function and get a new function.

Example

val multiplyByTwo = (_: Int) * 2
println(multiplyByTwo(5)) // Output: 10

8. Nested Functions

Functions can be defined inside other functions.

Example

def factorial(n: Int): Int = {
  def loop(acc: Int, n: Int): Int = if (n == 0) acc else loop(acc * n, n - 1)
  loop(1, n)
}
println(factorial(5)) // Output: 120

Conclusion

Functions in Scala offer a rich set of features, enabling concise and expressive coding styles. By understanding the different ways to define and manipulate functions, you can leverage the full power of functional programming in Scala.
